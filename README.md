## 📦 Features
- Full Inventory & Stock Management
- Real-time Sales Tracking (Cash & Online)
- Low Stock Alerts & Push Notifications
- User & Admin Roles (Secure Authentication)
- Progressive Web App (PWA) Installable on Desktop & Mobile
- 100% Fully Responsive Layout

## 🛠 Tech Stack
- Next.js 14 (App Router)
- React 18
- Vanilla CSS (Custom Design System)
- Prisma (PostgreSQL Database)
- Vercel Hosting
