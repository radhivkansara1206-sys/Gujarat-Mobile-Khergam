## 📦 Core Features
This application replaces traditional pen-and-paper or clunky excel sheets with a lightning-fast, tailored solution:

- **Full Inventory & Stock Management:** Track hundreds of mobile accessories by categories with real-time stock levels.
- **Advanced Sales Tracking:** Record cash, online (UPI), and gift transactions.
- **Defective Returns (Replacements):** Dedicated system to log defective items returned to distributors, complete with audit history.
- **Smart Notifications:** Automated low-stock warnings and real-time alerts when staff logs replacements.
- **Progressive Web App (PWA):** Installs directly onto desktops and mobile phones like a native application. Works offline.
- **100% Fully Responsive Layout:** Beautifully crafted UI that scales perfectly from the smallest phones to large desktop monitors.

## 🛠 Tech Stack & Architecture
Built using industry-leading, modern web technologies:
- **Frontend Framework:** Next.js 14 (App Router) & React 18
- **Styling:** Custom Vanilla CSS Design System (Zero bloat, sub-millisecond load times, vibrant glassmorphism UI)
- **Backend & Database:** Prisma ORM connected to a secure PostgreSQL Database
- **Hosting & CI/CD:** Vercel Global Edge Network with automated deployments

## 💡 Why this stack?
This stack was chosen to provide maximum performance. The application uses Server-Side Rendering (SSR) to load instantly, while the CSS is completely custom-written to ensure the UI is entirely unique, colorful, and highly interactive.
